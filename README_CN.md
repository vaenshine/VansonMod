## 📖 简介

**VansonMod** 是一款专为 **TrollStore (巨魔)** 环境设计的独立 Root Helper 工具。

与传统的越狱插件不同，它无需代码注入，而是利用 Mach 内核 API 在外部运行，因此 **内存调试功能无需越狱环境** 即可使用。核心集成**超强指针调试体系**、高性能内存调试器、应用存档管理、智能进程管理、专业 Hex 编辑器，同时为越狱用户提供完善的跨进程 RVA 调试套件；适配全尺寸 iOS 设备，完美支持原生 iPad 分屏，功能聚合式布局+极致交互优化，调试效率与稳定性双拉满，是 iOS 平台专业的应用调试工具。

### ⚠️ 兼容性提示
*   **内存调试 / 指针全功能 / 存档管理 / iPad分屏 / 图标切换**：✅ 支持 **所有** 设备（包括未越狱的巨魔用户）。
*   **RVA Patcher (汇编调试) / RVA调试记录管理**：⚠️ 需要 **越狱环境** (如 Dopamine, Palera1n) 才能正常工作。
    *   *原因:* 在未越狱的设备上，系统 AMFI 机制会严查代码签名。直接修改可执行代码段 (`__TEXT`) 会导致目标 App 立即闪退。

### ✨ 核心功能
*   **指针调试**:
    *   支持**手动/自动双模式指针搜索**，搭配专属**指针验证功能**，精准校验指针有效性，杜绝无效结果。
    *   支持指针地址**一键导入/导出**，调试结果可跨设备复用，大幅提升调试效率。
    *   新增指针专属配置开关，可对自动搜索规则**手动精细调参**，完美适配高阶用户深度调试需求。
    *   指针锁定、内存锁定、收藏夹**三合一聚合至同Tab页**，核心指针操作一键触达，无需多页面切换。
*   **内存调试器**:
    *   多模式搜索：精确、模糊、联合（高性能有序）、混合模式（二次筛选），检索效率全面优化。
    *   新增**数值精准筛选**：支持大于、小于、区间筛选三种模式，精准定位目标内存数据，调试更高效。
    *   临近搜索：基于现有结果扫描附近内存数据，自动去重，快速关联相关地址。
    *   批量调试：全选/范围选择修改、固定值/递增修改、数值锁定、无限滚动内存浏览全覆盖。
    *   智能配置：自定义浮点误差 (Float Tolerance)，适配各类调试场景参数需求。
*   **进程与应用管理**:
    *   智能过滤：自动识别用户及 TrollStore 应用、隐藏系统进程，支持按名称/BundleID/PID 快速筛选。
    *   多语言适配：智能显示 App 本地化名称，展示应用版本号，适配全语种设备。
    *   布局优化：**APP管理入口迁移至进程管理顶部**，核心操作集中化，无需层级跳转。
    *   交互升级：移除刷新图标，APP管理、进程选择、指针锁定、RVA管理四大模块**统一下拉刷新**，操作流畅无卡顿。
*   **应用存档管理**:
    *   一键备份/还原：完整备份 Documents/Library 目录、支持多存档，还原后自动结束目标进程即时生效。
    *   原生集成：系统「文件」App 直连管理，支持备份文件导入导出、全局备份统一查看与僵尸存档清理。
*   **🔧 跨进程 RVA 调试套件 (需越狱)【全面升级】**:
    *   实时调试：输入 Offset 和 Hex 自动 Patch 运行中代码，自动处理内存权限，内置常用指令预设。
    *   新增**RVA调试记录管理模块**：调试记录一键保存、快速导入/导出，核心调试数据高效留存，支持多记录批量管理。
*   **Hex 编辑器**:
    *   多视图切换：Hex/ASCII 同步分屏、仅Hex/仅文本视图自由切换，零边距视觉对齐，适配不同调试习惯。
    *   高效编辑：行编辑、地址快速跳转、同步滚动，修复翻页卡顿问题，操作流畅度拉满。
*   **个性化与体验优化**:
    *   图标自定义：新增**多款全新APP图标随心切换**，自由更换风格，增添使用趣味性。
    *   主题适配：支持跟随系统/浅色/深色三种主题模式，适配所有显示偏好。
    *   iPad专属：**原生iPad分屏完美适配**，支持屏幕自动旋转、台前调度模式稳定运行。
    *   稳定性提升：修复内存浏览卡顿、数值修改失效等已知BUG，整体运行稳定性大幅优化。
